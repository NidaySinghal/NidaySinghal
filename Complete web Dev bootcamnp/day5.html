Lists:-

This is an example of list we you can create lists using to tags one in "ul" tag and second is "ol" tag, "ul" tag stands for 'unordered list' and "ol" tag stands for 'ordered list'. The diffrence between these two tags are that "ul" tag does not represent any kind of sequence in it i.e., there are no numeric or alphabetical representation for progeression of listed items whereas, using "ol" tag you can represtent your items in a progressive pattern like numbers, alphabets, roman numbers etc. However, you can change the design of bullets that appear in "ul" tag into circle, square, disc etc.
To add items in your list you have to use "li" attributes in both "ul" and "ol" tags.

These is example of changed bullet design in unorderd lists that you can add using- "disc" attribute for filled black circle that will also appear without using this attribute because it is "ul" tag's preset, "square" attribute for filled square and "circle" attribute for unfilled circle. To add these design you have to add (type="disc"/"square"/"circle") attribute in "ul" tag.
