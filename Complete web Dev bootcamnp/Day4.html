Linking Images:-

This is "img" tag and "src", "alt" attribuites. "img" tag stands for 'image' and is used to link a image whereas, "src" attribute stands for 'source', in "src" attribute we enter the images's source(URL/File name with extension(only if it's in the same folder)) and "alt" attribute stands for 'Alternative Text', as the name suggests in "alt" attribute we enter text we want to appear if the image doesn't show up for ex. Adjusting image's dimensions

for adjusting dimensions(width,height) of our image we add "width", "height" attributes after "src" and "alt" attribuites, adding only "width" or only "height" attribute will automatically adjust the aspect ratio accordingly

Aspect ratio of image has disturbed because we enter both width and height in incorrect proportion

Aspect ratio is maintained because we have only entered width because of which aspect ratio has adjusted itself accordingly
