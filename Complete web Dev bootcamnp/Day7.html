"form" Tag:-

As the name suggest "form" tag is used to create forms in html. You should create your forms in "form" tag instead of freely taking inputs by user via diffrent input tags which are described below because there is an attribute called "action" attribute in "form" tag which will help you submitting the data entered by user in backend file for further processing, to add data in your backend file you have to type (action="your_backend_file.xyz") in "form" tag as done below

Taking iputs:-

For taking iputs from user you have to type "input" tag and attributes in it according to your choice of data for example for getting a text input you have to write (type="text") attribute in input tag. There are several types of inputs as described below.
